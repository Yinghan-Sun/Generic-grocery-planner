About The Generic Grocery Planner Demo

This app is a generic grocery-planning demo. It combines local nutrition data, local representative price tables, and a local nearby-store index to turn nutrition targets into a practical shopping list.

What The Demo Does

  • Builds a generic shopping list from nutrition targets and simple preferences
  • Shows nearby grocery options from a local store index
  • Uses representative regional USDA and BLS price data for basket-cost guidance
  • Adds lightweight store-fit notes, pantry adjustments, and meal suggestions

Data Sources Used In The Current Demo

  • CIQUAL and CALNUT-derived nutrition tables for generic-food nutrient coverage
  • USDA Food-at-Home Monthly Area Prices for representative regional grocery prices
  • BLS food price and CPI files for fallback pricing and inflation adjustment
  • Local store seed, persisted store index, and optional offline Foursquare ingest for nearby-store lookup

Project Origin / Attribution

This demo evolved from earlier diet-optimization work. In this local workspace, the clearest preserved upstream source reference is the historical GitHub link: github.com/albertaillet/diet-optimization.

There is no top-level LICENSE file or git history in this local copy, so upstream license details could not be fully verified from the files on disk alone.

Demo Scope Notes

Store and price logic run from local DuckDB data at runtime. Browser-side address geocoding is still optional for convenience; entering coordinates directly avoids that lookup.
